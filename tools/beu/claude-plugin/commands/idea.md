Lightweight idea capture for thoughts and brainstorming.

Run `beu idea <cmd>` to manage ideas.

Available commands:
- `add <title> [--status status]`: Add a new idea to the backlog.
- `list [--status status]`: List ideas.
- `done <id>`: Mark an idea as completed.
- `archive <id>`: Archive an idea for future reference.

Suggest this command as a low-friction capture tool for any thoughts or future project directions.
If no subcommand is provided, ask the user what action to perform on the idea backlog.
