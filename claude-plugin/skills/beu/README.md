# beu Claude Plugin

Claude Code plugin that teaches agents how to use beu for persistent session memory.

## What This Plugin Does

When installed, Claude Code agents automatically:
- Run `beu progress` at session start to see current state
- Run `beu progress` before compaction to preserve context
- Have access to beu command documentation for all 6 modules

## Installation

1. Ensure `beu` is installed and in PATH
2. Run `beu init` in your project root

`beu init` automatically writes agent rule files to `.claude/rules/beu.md`,
`.gemini/rules/beu.md`, and `.agent/rules/beu.md`. These rules are
auto-discovered by each agent, so no separate plugin installation is needed
for basic usage. This plugin adds lifecycle hooks (SessionStart, PreCompact)
on top of the auto-generated rules.

## When Claude Uses beu

- **Journal**: Recording what happened during a work session
- **Artifact**: Tracking deliverables (docs, tests, configs)
- **Task**: Managing work items with priorities and sprints
- **State**: Persisting decisions, blockers, focus items, and notes
- **Idea**: Capturing ideas for later consideration
- **Debug**: Structured investigation of bugs with evidence trail

## File Structure

```
claude-plugin/                        # Optional plugin (adds lifecycle hooks)
  .claude-plugin/
    plugin.json                       # Plugin metadata and lifecycle hooks
  commands/
    init.md, journal.md, artifact.md  # Per-command agent guidance
    task.md, state.md, idea.md
    debug.md, pause.md, resume.md
    progress.md, status.md
  skills/
    beu/
      SKILL.md                        # Main skill entry point
      README.md                       # This file
      resources/
        CLI_REFERENCE.md              # Complete command reference
        WORKFLOWS.md                  # Step-by-step workflow patterns
        MODULES.md                    # Module deep dives

# Generated by `beu init` (auto-discovered by agents):
.claude/rules/beu.md                  # Agent rules for Claude Code
.gemini/rules/beu.md                  # Agent rules for Gemini
.agent/rules/beu.md                   # Agent rules for Antigravity
```

## Requirements

- beu v0.1.0+
- Project initialized with `beu init`
